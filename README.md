# eMySQL

asynchronous erlang MySQL client.

## Compile with Rebar

compile rebar:

* git clone https://github.com/basho/rebar
* cd rebar
* ./bootstrap

compile emysql

* cp rebar emysql/
* cd emysql
*./rebar compile
